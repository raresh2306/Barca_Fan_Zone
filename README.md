# 🔴🔵 Barça Fan Zone

Welcome to the **Barça Fan Zone**! This is a full-stack web application dedicated to FC Barcelona fans. It serves as both an interactive fan portal and a fully functional e-commerce platform, allowing users to explore team history, player statistics, and purchase official club merchandise.

## ✨ Features

* **User Authentication:** Secure sign-up, login, and logout features for fans.
* **E-Commerce Store:** Browse team merchandise, manage a dynamic shopping cart, and submit orders safely.
* **Player Database:** Detailed biographies and profiles for both the men's and women's teams (e.g., Lamine Yamal, Pedri, Robert Lewandowski).
* **Match Tracking:** Keep up with the latest match histories and upcoming fixtures.
* **Multimedia Gallery:** High-quality images, videos, and historical moments from Camp Nou.
* **Responsive Design:** Fully mobile-friendly UI built with Bootstrap.

## 🛠️ Tech Stack

* **Frontend:** PHP, HTML5, CSS3 (Bootstrap, SASS), JavaScript (Vanilla)
* **Backend API:** Node.js (Express.js)
* **Database:** MySQL
* **Containerization:** Docker & Docker Compose
